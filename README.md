# LiftTrack

An AI-powered workout tracking mobile app built with Flutter and Node.js.

## Features
- Log your workout split and track weight progression per exercise
- Smart AI-generated notifications via Claude API — personalized to your recent performance
- JWT authentication with secure session management
- Clean, minimal dark UI designed for use at the gym

## Tech Stack
- **Mobile:** Flutter (Dart)
- **Backend:** Node.js + Express
- **Database:** MongoDB Atlas
- **AI:** Anthropic Claude API
- **Auth:** JWT
